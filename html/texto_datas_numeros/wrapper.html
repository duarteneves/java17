Wrapper

  • Classes que representam os tipos primitivos são chamadas de Wrapper Class.

  • Todas estas classes têm na sua API o método valueOf que aceita o respetivo tipo primitivo, ou uma string cujo valor seja o tipo primitivo, e devolve um objeto do tipo dessa classe.

  • O método xxxValue() permite, a partir de uma instância de uma Wrapper Class, obter o respetivo tipo.

  • Conversão auto-boxing de um tipo primitivo para Wrapper Class e conversão unboxing caso contrário. No entanto, apesar de facilitar a vida do programador deve evitar-se por questões de performance.

  • Podemos converter qualquer variável de tipo primitivo em um objecto do tipo String usando o método String.valueOf().

  • Qualquer Wrapper Class disponibiliza na sua API constantes que definem o valor mínimo e máximo que o respetivo objeto pode assumir.
