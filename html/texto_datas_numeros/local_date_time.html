Local Date / Time

  • A partir da versão 8, o Java disponibiliza o package java.time onde se incluem as classes LocalDate, LocalTime e LocalDateTime.

  • Substituir as classes java.util.Date e java.util.Calender.

  • Para obter uma instância de cada um do tipos anteriores invocar now() ou o método of() identificando os atributos específicos.

  • Podemos também obter a instância de objetos a partir de objetos de outro tipo através dos métodos atTime(), toLocalDate() ou toLocalTime().

  • Objetos imutáveis à semelhança os objetos String permite method chaining.

  • Invocação de métodos de forma sequencial sem serem necessárias variáveis intermédias.
