Map e HashMap

  • A classe HashMap é uma implementação da interface Map.

  • Uma instância de HashMap pode ser criada com o construtor default e neste caso fica com uma capacidade de 16 elementos.

  • Um HashMap aumenta a sua capacidade interna quanto tem essa necessidade.

  • Existe a possibilidade de definir um HashMap como uma determinada capacidade inicial.

  • Existe a possibilidade de definir um HashMap como uma determinada capacidade inicial e determinado load factor.

  • Um Map apenas de leitura pode ser criado usando o método Map.of(key, value, ...).

  • A classe HashMap é equivalente à classe legacy Hashtable, esta última é thread-safe porque é synchronized, e consequentemente mais lenta.
