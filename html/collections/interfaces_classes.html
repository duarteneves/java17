Interfaces e Classes

  • Presente no package java.util encontram-se definidas interfaces e classes da API.

  • Interfaces:

    • Iterable<T> - é a interface de nível mais alto e permite qualquer collection ser usada num ciclo forEach.
    • Collection<E> - estende de Iterable e descreve operações como adicionar e remover elementos.
    • List<E> - estende de Collection e descreve a list de elements indexada por int.
    • Set<E> - estende de Collection e descreve um conjunto de elementos únicos.
    • SortedSet<E> - estende de Set e apresenta esse conjunto ordenado.
    • Deque<E> - estende de Collection e descrever uma fila ligada permitindo comportamento FIFO e LIFO.
    • Map<K, V> - contém um Set de chaves únicas e uma List de valores.

  • Classes:

    • ArrayList<T> - imoplementa a interface List.
    • HashSet<E> - implmenta a interface Set.
    • TreeSet<E> - implmenta a interface SortedSet.
    • ArrayDeque<E> - implmenta a interface Deque.
    • HashMap<K, V> - implementa a interface Map.

  • Estas serão as interfaes / classes de coleções / conjuntos mais usadas.
