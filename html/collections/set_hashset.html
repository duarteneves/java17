Set e HashSet

  • A classe HashSet é uma implementação da interface Set.

  • Um HashSet pode ser instanciado com o construtor default e nesse momento é criado um set com capacidade para 16 elementos e load factor 0.75.

  • Um HashSet aumenta a sua capacidade interna quanto tem essa necessidade.

  • Existe a possibilidade de definir um HashSet com uma determinada capacidade inicial.

  • Existe a possibilidade de definir um HashSet com uma determinada capacidade inicial e determinado load factor.

  • Um Set apenas de leitura pode ser criado usando o método Set.of(<T>...).

  • Uma List permite elementos duplicados, enquanto que um Set não permite. Quando se transforma uma List num Set, os elementos duplicados são descartados.
