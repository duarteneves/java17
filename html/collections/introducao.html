Introdução

  • A Java API disponibiliza um conjunto de classes que permitem gerir coleções de objetos.

  • Esta API é mais rica e flexível do que o uso de arrays.

  • As classes desta API permitem:

    • o uso de genéricos
    • aumentar a capacidade de acordo com a necessidade
    • integrar algoritmos de pesquisa
    • validar os elementos, por exemplo, verificar a sua unicidade
    • integrar algoritmos de ordenação
    • operações concurrentes (treahd-safe)
    • iterar sobre os elementos da coleção
