Interfaces funcionais

  • Interfaces que definem apenas um método (public, abstract).

Problema

  • Não é possível implementar apenas alguns métodos definidos numa interface.

  • Uma interface com muito métodos abstratos é um problema. A classe que implementa esta interface tem de implementar todos os seus métodos, mesmo quando em determinado contexto, não faça uso desses métodos.

Solução

  • Uma classe pode implementar quantas interfaces forem necessárias.

  • Uma interface pode definir apenas um método abstrato.

  • É uma abordagem flexível e recomendada no desenho de interfaces.
