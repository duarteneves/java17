var

  • A partir da versão 10 do Java passou a ser possível usar var (type inference) à semelhança do Javascript.

  • Não há a necessidade de descrever o tipo da variável se esta não for ambígua perante determinado contexto.

    • Inferir o tipo da variável quando é inicializada.
    • Não é necessário identificar o tipo da variável quando esta é afetada por outra variável.

  • Não é suportado o uso de var nos atributos de uma classe e no tipo de retorno de um método.

  • A variável var tem de ser imediatamente atribuida (assigned).

  • Não é possível inicializar uma variável var com null.

  • Não é possível declarar multiplas variáveis.

  • Não pode ser reassigned e polimorfismo não é suportado.

  • O uso do var permite uma melhor leitura de código.

  • A palavra var n é uma keyword, mas sim um identificado especial, ver slide Keywords.
