Ciclos

  • Os ciclos em Java contêm os seguintes componentes:

    • declaração do iterador
    • condição de fim de ciclo
    • iterador, tipicamente esse valor é incrementado / decrementado
    • o corpo do ciclo

  • Iteração - mecanismo usado para passar de elemento em elemento, por exemplo, para percorrer uma string ou um array.

  • Iterador - cursor usado para aceder de forma sequencial a elementos de uma coleção.

  • Tipos de ciclo: while, do while, for e forEach.

    • while - disponibiliza um iterador simples
    • do while - garante de o corpo definido no ciclo é executado pelo menos uma vez
    • for - mantém a declaração, a condição de paragem e o iterador juntos, separados por ";"
    • forEach - disponibiliza um modo mais conviniente de iteração sobre um array ou uma collection

  • forEach não é um operador nem uma palavra reservada, apenas descreve um ciclo for que itera sobre uma string, um array ou uma collection sem ser necessário manter um index ou uma lógica de iteração.
