Arrays []

  • Um array é uma estrutura de elementos do mesmo tipo de tamanho previamente definido.

  • O acesso aos elementos faz-se através da identificação de índices.

  • A declaração de um array determina o tipo de elementos que serão guardados.

  • A criação de um array determina o comprimento do array, ou seja, o nº de elementos.

    • Uma vez criado o tamanho de um array não pode ser alterado
    • Num array de tipos primitivos todos os elementos são iniciados com 0, false ou ' ', dependendo do tipo primitivo.
    • Num array de objetos todos os valores assumem a referência null.

  • Inicializar um valor do array efetua-se usando [] e o índice da posição no array que pretendemos atribuir.

    • O índice começa em 0 e termina em length - 1.
  • O acesso a um elemento de um array efetua-se usando [] e o índice da posição no array que pretendemos obter.
