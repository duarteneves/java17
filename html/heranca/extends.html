Extends Class

  • Todas as classes descendem da classe java.lang.Object.

  • A classe Object define um conjunto de operações que todas as classes Java herdam e utilizam.

  • A classe Pai é conhecidda como superclass.

  • A classe Filho é conhecida como subclass.

  • Uma classe pode apenas estender de uma outra classe. Herança múltipla não é permitida em Java.
