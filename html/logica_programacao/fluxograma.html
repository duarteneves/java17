Exemplo 3 - Somar 3 números

  • dados de entrada (input): números, por exemplo, os algarismos 1, 2 e 3

  • lógica ou algoritmo: fórmula de cálculo, neste caso usar o operador de soma (+) que já conheces dos tempos da primária

  • dados de saída (output): soma

  • cenários de teste: 1, 2, 3 -> 6

Exemplo 4 - Somar 3 números, considerando apenas os que são pares

  • dados de entrada (input): números, por exemplo, os algarismos 1, 2 e 3

  • lógica ou algoritmo: fórmula de cálculo, neste caso usar o operador de soma (+), mas aplicar apenas quando os dois algarismos são número par

  • dados de saída (output): soma

  • cenários de teste: 2 -> 2
