Modificadores de acesso de uma Classe

  • public - visível para outras classes.

  • protected - visível para classes no mesmo package ou sub classes.

  • <default> - visível apenas para classes no mesmo package.

  • private - visível apenas dentro da classe.

  • O mais usual é usar o private para parâmetros de uma classe ou protected para que seja possível o acesso das classes filho. A definição public usa-se geralmente na declaração de classes, construtores e de constantes.
