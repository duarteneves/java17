Acesso a outras classes

  • As classes acedem a outras classes dentro do mesmo package.

  • Para aceder a uma classe noutro package é necessário identificar como prefixo da class o package onde ela se encontra.

  • Usar o import entre a definição de package e a declaração da classe.

  • Todas as classes importam por default o package java.lang.*.

  • O import apenas serve o momento de compilação, não tem efeito no momento de execução.
