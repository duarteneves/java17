Método Main

  • O nome do método tem de ser main.

  • O método tem de ser declarado como public para que seja possível invocar fora da classe.

  • Tem de ser static para não ser necessário instância da classe onde está definido.

  • O método retorna void, ou seja, não retorna qualquer valor.

  • Tem apenas um parâmetro como argmento e esse parãmetro é um array de String. O nome do argumento é irrelevante.
