Comentários

  • Existem 3 tipos de comentários: single line, multi-line e javadoc.

  • Comentários podem ser adicionados em qualquer momento no código fonte da aplicação Java.

  • Javadoc deve ser definido na declaração de uma classe, método e parâmteros.

  • Empresas definem como obrigatório a existência de Javadoc ao nível das classes e métodos, exceto métodos getter e setter.

  • Toda a API JDK está documentada usando Javadoc.

  • Javadoc pode ser ainda usado para elaboração de documentação para o cliente, por exemplo.
