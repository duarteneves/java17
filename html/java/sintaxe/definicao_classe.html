Definição de uma Classe

  • Uma determinada class está num ficheiro com extensão java e com o mesmo nome da class.

  • As classes são agrupadas em packages. O default package é uma má prática. Nome de uma class é composto por package + class.

  • Os packages são pastas onde os ficheiros java são guardados.

  • O nome dos packages são geralmente definidos pelo domínio da empresa, por exemplo com.oracle ou org.apache .
