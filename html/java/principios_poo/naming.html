Naming

  • Java é case sensitive.

  • A classe deve ser um substantivo ou um nome e pode ser compostos por várias palavras. Cada palavra deve começar por letra maiúscula.

  • Variáveis devem começar com letra minúscula, mas as restantes palavras podem começar com maiúscula. Não devem começar com número ou _.

  • Constantes devem ser escritas apenas com maiúsculas e cada palavra deve ser separada por _.

  • Método deve começar por um verbo, começar com letra minúscula e as restantes palavras com letra maiúscula.

  • Uso de sufixos e prefixos em classes, métodos e variáveis.
