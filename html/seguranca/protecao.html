Proteção de dados

  • Proteger dados sensíveis como identficiadores de utilizadores e clientes, moradas, emails, nº de contas ou cartões de crétido, palavras chaves.

  • Persistir os dados cifrando essa informação e obter os dados de forma mascarada.

  • Não manter estes dados em memória.

  • Não indicar este género de informação no lançamento de exceções ou mensagens de logging.

  • Para cifrar ou decifrar (encriptar/ desencriptar) o Java disponibiza uma API presente no package javax.crypto.

  • São suportados diferentes algoritmos de cifra, por exemplo AES (advanced Encryption Standard),
