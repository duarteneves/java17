JavaScript (JS)

  • Da mesma forma, também é possível injetar código javascript nos inputs de um formulário, ou através do url. O código pode ser persistido ou passar apenas pelo servidor e sempre que uma determinada página é apresentada esse código pode ser executado.

  • A solução passa por validar e sanitizar todos os valores que são passados através do browser. As validações podem ser feitas no frontend, mas devem estar sempre presentes no backend.

  • A API Java não suporta este género de validação, no entanto, aconcelha o uso de bibliotecas open source como a bibliteca disponívela pela OWASP.
