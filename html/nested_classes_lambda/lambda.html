Expressões lambda

  • A implementação inline de interfaces pode ser ainda mais fácil se usarmos expressões lambda.

  • As propriedades da expressão lambda são inferidas a partir do contexto onde é usada:

    • o contexto de invocação infere que interface funcional é implementada.
    • o método a implementar pela expressão lambda é o único método abstrato da interface funcional.
    • são inferidos os parâmetros que o método deve receber.
    • o tipo de retorno infere a declaração que é retornada.

  • A expressão lambda apenas deve conter o nome dos parâmetros e o algoritmo.

  • O símbolo -> separa os parâmetros da implementação.
