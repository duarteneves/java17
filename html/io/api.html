API

  • As classes de IO encontram-se definidas nos seguintes packages java.io e java.nio.

  • As classes abstratas definem métodos de leitura de dados de texto e binário.

  • As classes concretas, descendentes das classes abstratas, disponibilizam tipos para melhor maniupular streams de IO, nomeadamente:

    • ligar a fontes (inputs) e destinos (outputs)
    • transformar conteúdo
    • execução de conteúdo em buffers
    • disponibilzar métodos de conveniência

  • Além destas classes existem outras na API Java.
