Scanner

  • A classe System lê e escreve, de e para, os dispositivos padrão do sistema operativo. Designados por standard input e standard output, por omissão, representam respetivamente, o teclado e o ecrã.

  • A class System disponibiliza atributos de input, output e output de erro, através das seguintes constantes:

    • System.in - InputStream (binário)
    • System.out - PrintStream (binário)
    • System.err - PrintStream (binário)

  • A class java.util.Scanner disponibiliza API para tratar dados de input.

  • Não é recomendado o uso dos padrões de input e output. Estes padrões representam acesso sequencial a recursos, o que tem um impacto significativo na performance e escalabilidade do programa.
