Serialização

  • Os objetivos da serialização:

    • Serialização é o processo de escrita de objetos em memória para uma stream.
    • Desserialização é o processo de leitura de objetos a partir de uma stream.

  • Quando é usada serialização.

    • enviar objetos pela net
    • escrita de objetos em ficheiros

  • A serialização não é uma solução adequada para armazenar dados a longo prazo.

    • E se o código se alterar? Se a classe se alterar? É possível ler a partir de de algo guardado numa vesão aanterior?
    • O conteúdo serializado é específico da compilação do código naquele momento.
