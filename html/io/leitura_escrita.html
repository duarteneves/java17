Leitura / Escrita

  • A API Java disponibiliza as seguintes classes para leitura e escrita de objetos em streams.

    • ObjectOutputStream - escrita de objetos serializáveis para uma stream.
    • ObjectInputStream - leitura de objetos serilizáveis de uma stream.
