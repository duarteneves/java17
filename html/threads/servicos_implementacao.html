Implementação

  • Para implementação de serviços podemos optar por um de dois tipos: Runnable ou Callable

  • Os seviços runnable são implementados com a definição do método public void run() da interface Runnable. Sendo que para executar o serviço a API da interface ExecutorService disponibiliza os métodos execute ou submit.

  • Os serviços callable são implementados através da definição do método public <T> call() da interface Callable.

  • O método submit, como o nome indica submete uma tarefa para execução. Retorna um objeeto Future que representa os resutlados de execução daquela tarefa.
