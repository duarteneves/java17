Lock

  • O Java disponibiliza classes do package java.util.concurrent.locks que permitem, como o nome indica, fazer lock no momento em que um bloco de código é executado.

  • O lock de leitura previne que outras threads acedam de forma concorrente e alterem o estado do objeto. São permitidos acessos para leitura.

  • O lock de escrita previne que outras threads acedam de forma concorrente e alterem o estado do objeto.
