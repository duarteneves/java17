Serviços

  • A classe java.util.concurrent.Executors disponibiliza um número de gestores de threads que nos permitem criar serviços de acordo com determinadas caraterísticas:

    • Fixed Thread Pool - reutiliza um nº fixo de threads
    • Work Stealing Pool - mantém o nº de threads suficiente para suportar um nível de paralelismo
    • Single Thread Executor - usa uma única thread
    • Cached Thread Pool - criar novas threads de acordo com a necessidade ou reutiliza as threads existentes
    • Scheduled Thread Pool - programa tarefas para serem executadas periodicamente ou a partir de determinado tempo
    • Single Thread Scheduler Pool - programa tarefas para serem executadas usando apenas uma única thread
    • Unconfigurable executor Service - disponibiliza um modo para "congelar" outra configuração de serviço
