O que é um módulo?

  • Um módulo é uma agregação de alto nível que compreende um ou mais módulos e outros recursos como imagens, por exemplo.

  • Uma aplicação definda por módulos deve ter definido um ficheiro module-info.java presente na root da definição de cada módulo. Este ficheiro pode ter definidas as seguintes propriedades:

    • nome do módulo - deve único, por exemplo com.yourcompany.whatever
    • requires - dependências necessárias, identificação de outros módulos de que este módulo depende
    • exports - packages que o módulo exporta, tornando-os disponíveis para outros módulos
    • opens - identificar packages que estão acessíveis via reflection
    • uses - serviços usados pelo módulo disponíveis por outros módulos
    • provides - serviços disponibilizados pelo módulos e a respetiva implementação
