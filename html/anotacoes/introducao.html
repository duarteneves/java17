Introdução

  • Anotações começam por @.

  • As anotações podem ser usadas em tempo de compilação, processo de deploy e em runtime.

  • Anotações ajudam a associar meta informação aos elementos da aplicação, tais como, classes, variáveis, construtores e métodos.

  • Anotações podem ser usadas para definir parametrizações sendo assim uma alternativa ao XML.

  • A Java API disponibiliza anotações a partir dos packages java.lang e java.lang.annotation.

  • O Java permite a definição de novas anotações.
