Interfaces funcionais

  • Interfaces funcionais definidas no package java.util.function .

  • Existem 5 tipos e interfaces usadas pelos métodos stream.

    • Predicate<T> - define o método boolean test(T t) para aplicar condições e filtrar elementos.
    • Function<T, R> - define o método R apply(T t) para converter tipos de elementos.
    • UnaryOperator<T, R> - uma variante da Function, define o método T apply(T t) para converter valores.
    • Consumer<T, R> - define o método void accept(T t) para processar elementos.
    • Supplier<T, R> - define o método R get(T t) para produzir elementos.
