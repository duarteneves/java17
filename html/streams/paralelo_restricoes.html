Restrições do processamento paralelo

  • Não definir operações que necessitem de acesso sequencial a um recurso partilhado.

  • Não definir operações que alteem um recurso partilhado.

  • Usar métodos da API Collectors corretamente: toMap (sequencial), toConcurrentMap (paralelo).

  • Aplicar processamento paralelo para um número considerável de elementos.

  • Se estiverem disponívels múltiplos CPUs.

  • Se os CPUs disponíveis tiverem alta performance.
