Statements

  • A interface java.sql.Statement representa todos os tipos de instruções SQL.

  • Uma vez obtida a connection e a partir da connection o bjeto da interface Statement a API permite invocar instruções: select, insert, update, delete, create, alter e drop.

  • A utilização de statements tem desvantagens quando comparadas com as prepared e callable statements que vimos anteriormente.

    • permitem concatenação de queries o que aumenta o risco de SQL Injection
    • as instruções têm de ser parsed e recompiladas antes de cada execução
