Result Set

  • A interface java.sql.Result permite iterar e obter os resutlado da resposta SQL.

  • Através do método next() podemos iterar o conjunto de resutlados. Este método retorna true se ainda existirem resultados e false caso já não existam. Quando o valor retornado é true o apontador muda para o próximo objeto, este comportamento permite aplicar as seguintes operações sobre esse elemento.

  • Para obter os valores de cada coluna, o programador pode aplicar um dos seguintes métodos:

    • getXXX(column_position)
    • getXXX(column_name)
    • getObject(column_position, class)
    • getObject(column_name, class)

  • A primeira posição começa com o valor 1.
