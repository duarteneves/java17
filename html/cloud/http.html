HTTP

  • Os métodos mais comuns de pedidos HTTP são GET (default), POST, PUT e DELETE.

  • As respostas HTTP contêm sempre o status code que nos informa a natureza da resposta, por exemplo:

    • 200 -> OK
    • 404 -> Not Found
    • 500 -> Internal Server Error

  • Os pedidos e respostas HTTP contêm headers que descrevem meta informação, por exemplo:

    • content type - text/html, multiparte/form-data
    • length
    • character encoding - UTF-8
    • date and time

  • Os pedidos e respostas contêm dados, estes dados são encapsulados pelo body.
