Package e Deploy

  • Java EE:

    • enterprise archives (ear), web archives (war) e java archives (jar)
    • Java EE server que corre sobre a sua JVM
    • múltiplas aplicações podem ser deployed sobre o mesmo servidor Java EE

  • Java SE / MP:

    • java archives (jar)
    • pode ser packaged a partir de Java Image
    • server arranca a partir do código da aplicação
    • cada aplicação corre sobre a sua JVM

  • O Java recomenda o uso de Maven para automatizar e gerir o deployment da aplicação Java.
