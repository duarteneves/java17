Introdução

  • Trabalhar na cloud está na moda e as equipas de desenvolvimento estão, tendencialemnte, a virar-se para este tipo de desenvolvimento.

  • O cloud computing é um processamento baseado numa série de servidores espalhos pela internet. De acordo com o seu processamento cada servidor pode partilhar os seus recursos com outros servidores.

  • Ferramentas como o Gmail, DropBox ou Google drive são exemplos de trabalhar na cloud. O desenvolvimento de sites / CMSs integrados nos servidores de hosting é outro exemplo.

  • Trabalhar na cloud tem desafios, mas também algumas vantagens muito importantes:

    • alta disponibilidade
    • escalabilidade e desempenho
    • backup
    • atualizações
    • control de custos

  • Existem 3 modelos de implementação na cloud:

    • pública
    • privada
    • híbrida

  • Existem várias plataformas para deploy de aplicações Java na cloud:

    • Oracle Java Cloud Service
    • Amazon Web Services (AWS)
    • Google App Engine
    • Heroku
    • Cloudfoundry
